# Retail Sales Analysis (R)
## Project Overview
This project analyzes an online retail dataset using R to uncover sales trends, top-performing products, and country-level performance.
## Tools Used
- R
- tidyverse (dplyr, ggplot2)
- lubridate
## Key Analyses
- Data cleaning and preparation
- Monthly and yearly sales trends
- Sales by country
- Top-selling products
## Key Insights
- Sales peak during certain months of the year
- A small number of countries contribute most revenue
- Top products drive a large portion of total sales
## Dataset
The dataset used is the Online Retail dataset.
(Source: UCI Machine Learning Repository)
